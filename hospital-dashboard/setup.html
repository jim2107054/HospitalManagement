1Database Setup

Create a MySQL database and import the schema:

mysql -u root -p
CREATE DATABASE hospital_management;
USE hospital_management;
SOURCE database.sql;

2Configure Database Connection

Edit the database settings in includes/database.php:

private $host = 'localhost';
private $db_name = 'hospital_management';
private $username = 'your_username';
private $password = 'your_password';

3Verify Installation

Check that these components are working:

  • Web server is running (Apache/Nginx)
  • PHP is configured and working
  • MySQL database is accessible
  • All project files are in the web directory

4Access the Dashboard

Open your web browser and navigate to:

http://localhost/hospital-dashboard/
